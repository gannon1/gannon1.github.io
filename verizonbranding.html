Project Summary

This goal of this project was to create new branding for a company. I chose to rebrand Verizon. I was tasked with creating a webpage, mobile app layout, banners and billboard advertisements

Skills Learned

  • Skill with Adobe Photoshop and Illustrator
  • Skill with creating layout

Challenges

The only challenge I had with this project was researching the brand that I was creating my designs for. This required examining Verizon's website, understanding the way they want to present their business and figuring out a design that would put Verizon's best face forward.
